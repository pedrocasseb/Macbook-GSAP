# 🍎 MacBook M4 Landing Page

Uma landing page moderna e animada inspirada no **MacBook M4**, desenvolvida com **React**, **Tailwind CSS** e **GSAP**.  
O projeto foi criado com foco em performance, animações fluidas e uma experiência visual premium.

---

## ✨ Tecnologias Utilizadas

- ⚛️ React
- 🎨 Tailwind CSS
- 🎬 GSAP
- ⚡ Vite
- 🖥️ JavaScript

---

## 🚀 Funcionalidades

- Animações suaves com GSAP
- Layout totalmente responsivo
- Design inspirado no visual da Apple
- Transições modernas e fluidas
- Seções animadas no scroll
- Interface minimalista e elegante
